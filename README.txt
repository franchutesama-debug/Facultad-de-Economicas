Este directorio contiene los archivos necesarios para publicar el diagrama de correlatividades de la carrera de Actuario con materias optativas y nodos fijos.

Para publicarlo como una página web:
1. Copia todo el contenido de esta carpeta (index.html y cualquier otro archivo) a un servidor web o un servicio de alojamiento estático (por ejemplo, GitHub Pages, Netlify, etc.).
2. Asegúrate de que la ruta al archivo index.html sea accesible públicamente, por ejemplo: https://tudominio.com/index.html.
3. El archivo index.html incluye todo el código necesario y cargará el diagrama interactivo en cualquier navegador moderno.

Una vez que tengas la URL pública, puedes generar un código QR con esa dirección para compartir el diagrama.
